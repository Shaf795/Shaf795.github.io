Finite Element & Computational Tools

ANSYS APDL

Extensive experience in parametric modeling and scripting using ANSYS APDL for stress analysis and fracture mechanics applications. Used for developing reusable scripts, conducting mesh refinement studies near singularities, and extracting fracture parameters such as stress intensity factors.

ABAQUS with Python Scripting

Experience in finite element modeling using ABAQUS, complemented by Python scripting for automation, batch simulations, and post-processing. Applied to crack modeling, mixed-mode loading studies, and parametric investigations in fracture mechanics.

MATLAB

Utilized for numerical computation, analytical validation, data processing, and visualization. MATLAB is used to support theoretical derivations, validate FEM results, and develop custom post-processing routines.
